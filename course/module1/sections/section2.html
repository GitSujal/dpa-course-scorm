Data Types and Sources

Understanding Data Types

Types of Data

Data comes in various forms, each serving different purposes and requiring different handling approaches:

  • Structured Data (databases, spreadsheets)
  • Unstructured Data (text, images, videos)
  • Semi-structured Data (JSON, XML)
  • Time-series Data (sensor readings, stock prices)

Common Data Sources

In today's digital landscape, data flows from numerous sources, each providing unique insights:

  • Customer Relationship Management (CRM) systems
  • Enterprise Resource Planning (ERP) software
  • Internet of Things (IoT) devices
  • Social media platforms
  • Web analytics tools
